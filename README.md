# fafa's-tts

一款只提供必要体验的 Chrome 插件。

功能：选中文本后右键朗读，基于Azure TTS API，使用 Azure 语音合成（Speech Service）。

## 使用方式

1. 打开 Chrome -> 扩展程序 -> 开启开发者模式。
2. 点击“加载已解压的扩展程序”，选择此目录。
3. 进入扩展设置页，填写 API Key、区域、语速和声音。
4. 在网页中选中文本，右键选择“朗读”，或点击扩展图标朗读。
5. 点击插件图标可打断朗读。
6. 右键扩展图标可直接打开设置页。
7. 保存最近的朗读的声音，可重复播放或下载，不浪费额度。

## 说明

- Azure 语音合成请求地址：`https://{region}.tts.speech.microsoft.com/cognitiveservices/v1`
- 缓存基于文本 + 声音 + 语速 + 区域，默认保留 50 条，可在设置页查看/播放/下载。
